gf.AssetLoader Class

Uses
  • EventEmitter
Extends Object
Defined in: src\loader\Loader.js:10

The AssetLoader loads and parses different game assets, such as sounds, textures, TMX World files (exported from the Tiled Editor), and Spritesheet files (published from Texture Packer).
