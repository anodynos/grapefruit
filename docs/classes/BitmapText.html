BitmapText Class

Extends Container
Defined in: src\text\BitmapText.js:11

A Text Object will create (a) line(s) of text using bitmap font. To split a line you can use "\n", "\r" or "\r\n" You can generate the fnt files using bmfont for windows or bmglyph for mac.

  • font
  • [style]
)

Defined in src\text\BitmapText.js:11

Parameters:

    The copy that you would like the text to display

  • font Object

    The font data object (this is generally grabbed from game.cache.getBitmapFont('mykey'));

    • name String

      The name of the font

    • size Number

      The base size of the font

    • lineHeight Number

      The line height of the font

    • chars Object

      The characters in the font, each should have a texture and kerning info

  • [style] Object optional

    The style parameters

    • [size=null] Number optional

      The font size of the text, overrides the font's size

    • [align="left"] String optional

      An alignment of the multiline text ("left", "center" or "right")
