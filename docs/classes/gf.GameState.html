gf.GameState Class

Extends Container
Defined in: src\game\State.js:9

GameStates are containers that represent different states of a game

Constructor

gf.GameState

(
  • [name]
  • [settings]
)

Defined in src\game\State.js:9

Parameters:

  • [name] String optional

    The name of this state

  • [settings] Object optional

    All the settings for this game state

    • [gravity] Number optional

      The gravity constant for the physics system (default is 9.87, which is normal Earth gravity)

Example:

 var state = new GameState(game, 'battle');
 state.addChild(battlePlayer);
 state.addChild(enemy);

 game.enableState(state); //or you can use the name from the ctor 'battle'
