gf.Sprite Class

Uses
  • EventEmitter
Extends PIXI.Sprite
Defined in: src\display\Sprite.js:10

The base Sprite class. This class is the base for all images on the screen

Constructor

gf.Sprite

(
  • textures
  • [speed]
  • [start]
)

Defined in src\display\Sprite.js:10

Parameters:

  • textures Texture | Array | Object

    The texture for the sprite to display, an array of texture to animation through, or an animation object. The later looks like: { animationName: { frames: [frame1, frame2], rate: 2 } } where each frame is a Texture object

  • [speed] Number optional

    The speed of the animations (can be overriden on a specific animations)

  • [start] String optional

    The animation to start with, defaults to the first found key otherwise

Example:

 var spr = new gf.Sprite(texture);

    The sprite that you collide with

    Collision data object that contains information about the intersection
