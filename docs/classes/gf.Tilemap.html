gf.Tilemap Class

Extends Container
Defined in: src\tilemap\Tilemap.js:10

Tiled map, expects a Tiled TMX file loaded by the loader as the argument. The loader knows to load all textures and other resources when loading a world TMX file, and this expets that to already be done.
